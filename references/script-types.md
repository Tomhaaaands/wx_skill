# 脚本类型选型指南

6 种脚本类型，生成时先按决策树定 `script_type`，再定中段段落功能与段数。

## 选型决策树

```
脚本核心诉求？
├── 解决某痛点 / 功能性产品 → pain_point（痛点解决型）
├── 日常高频使用、生活场景 → scene（场景植入型）
├── 效果可量化、可对比 → unboxing（开箱测评型）
├── 需教育市场 / 干货输出 → testimonial（口播种草型）或 tutorial（教程/制作型）
└── 品牌 / 情感 / 关系共鸣 → drama（剧情种草型）
```

## 6 类型对照表

| script_type | 中文名 | 建议中段数 | 中段段落功能（参考命名，按顺序） | 适用场景 |
|------|------|------|------|------|
| `pain_point` | 痛点解决型 | 3 | 痛点放大 / 产品出场 / 效果展示 | 功能性产品、解决明确痛点 |
| `scene` | 场景植入型 | 2 | 生活场景还原 / 产品自然融入 | 日常高频、生活化品类 |
| `drama` | 剧情种草型 | 4 | 剧情设定 / 冲突发展 / 反转高潮 / 产品融入 | 情感共鸣、关系代入 |
| `testimonial` | 口播种草型 | 4 | 身份锚定 / 干货输出 / 产品推荐 / 信任背书 | 需专家/达人背书、教育市场 |
| `unboxing` | 开箱测评型 | 4 | 开箱过程 / 外观展示 / 功能测试 / 使用体验 | 新品、效果可量化 |
| `tutorial` | 教程/制作型 | 3（≤7） | 产品/原料出场 / 制作过程 / 效果展示 | 教程、制作、对比 |

> 中段段数上限：**教程/制作型 ≤7，其余 ≤5**；多余细步骤合并到相邻段落功能下，不必每步一段。

## 各类型爆款要点

- **痛点解决型**：Hook 必须在前 3 秒戳到真实痛点；中段「痛点放大」要给具体证据（数字、画面、案例），不能空喊；产品出场要承接痛点，给出解法。
- **场景植入型**：先 1–2 个真实生活镜头建立代入感，产品作为「顺手用的工具」自然露出，不硬广；CTA 用「我也在用」式口吻。
- **剧情种草型**：冲突要尖锐、反转要意外；产品在反转后出现，承担「破局」角色；台词生活化、忌说教。
- **口播种草型**：开头先用身份锚定建立信任（我是谁、为什么我有发言权）；干货占大头，产品作为「我的私藏推荐」自然带出。
- **开箱测评型**：全程第一人称体验视角；功能测试要可感知、有反馈；缺点也可提一两个增加可信度。
- **教程/制作型**：步骤清晰、节奏快切；成品展示要诱人；产品露在制作过程中而非生硬插入。
