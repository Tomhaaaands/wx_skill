# 合规与风格铁律

生成脚本必须过本规则；不达标回炉重写，最多 3 轮。

## 1. 极限词扫描（命中即替换）

| 类别 | 词例 |
|------|------|
| 绝对化 | 最 / 第一 / 顶级 / 顶级 / 绝对 / 100% / 永远 / 必定 |
| 夸大 | 封神 / 绝了 / OMG / 神器 / 完爆 / 秒杀一切 / 天花板 |
| 引导极限 | 史无前例 / 全网最低 / 抢疯了 / 再不抢就没了 |
| 权威背书造假 | 国家级 / 御用 / 专供（无授权不得用） |

替换原则：极限词 → 客观、口语化表述（如「最干净」→「洗得挺干净」、「100% 有效」→「我自己用下来挺管用」）。

## 2. 功效宣称红线（医疗 / 未经证实功效）

- 禁止宣称治疗、治愈、根治、药效（治病 / 治愈 / 根治 / 疗效 / 药到病除）。
- 食品 / 日化不得宣称医疗功效；保健品类按实际审批功效表述，不超范围。
- 未经证实的功效（美白 / 抗衰 / 减肥 / 壮阳等）需有资质或改用「我个人感受是…」「用一段时间感觉…」式主观体验表达。

## 3. 价格合规

- 出现价格必须带规格与单位（如「¥29.9 / 3 袋」「￥9.9 一瓶 500ml」），不得裸价。
- 不用「全网最低」「跳楼价」「亏本甩」等极限价格用语。
- 促销机制（满减/赠品）需具体可兑现，不夸大。

## 4. 口语化风格铁律

- **短句为主，每句 ≤15 字**；长句拆短。
- 用「...」标注口播停顿位置，模拟真实说话节奏。
- **不用书面连接词**：此外 / 因此 / 综上所述 / 不仅…而且 / 然而。
- 拒绝广告腔：不说「家人们」「宝宝们赶紧下单」「真的绝绝子」。
- 像跟朋友聊天，自然口语；可适度口语词（那 / 其实 / 跟你说），忌书面腔。
- 产品植入自然：用「我家一直用」「我给你看一下我们家厨房」式带入，不生硬推销。

## 5. 产品缺失时的降级

- 无产品 → `product_conversion=null`，脚本走纯种草 / 涨粉 / 知识分享向，CTA 用关注 / 收藏 / 评论互动，不出现购买引导。
