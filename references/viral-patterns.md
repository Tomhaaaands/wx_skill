# 爆款模式库（保证爆款潜质）

生成脚本时按本库选钩子 / CTA / 节奏 / 情绪曲线；兜底生成**必须**通过「爆款潜质自检清单」≥3 项。

## 钩子类型（hook.type）

| 类型 | 手法 | 适用 |
|------|------|------|
| 痛点提问 | 抛出受众真实痛点提问 | 痛点解决型、科普型 |
| 利益 | 开门见山给利益/结果 | 教程、好物种草 |
| 冲突 | 制造对立或意外 | 剧情种草、对比 |
| 悬念 | 留悬念引导看完 | 各类型通用 |
| 反差 | 制造常识反差 | 测评、科普 |
| 共鸣 | 直击情绪共鸣点 | 情感、剧情 |
| 数据 | 用数字/数据震撼 | 科普、测评 |
| 反常识 | 打破既有认知 | 科普、种草 |
| 场景建立 | 用真实生活场景代入 | 场景植入型 |
| 身份锚定 | 用身份建立信任 | 口播种草型 |

> Hook 必须在前 3 秒命中至少 1 种钩子手法。

## CTA 类型（cta.type）

收藏 / 关注 / 购买 / 点击购物车 / 评论互动 / 转发 / 点赞。

- 带货脚本 → 购买 / 点击购物车；种草涨粉 → 关注 / 收藏；互动向 → 评论互动。
- CTA 要自然，避免「赶紧下单吧」式生硬推销；用「我也在用」「能把握的尽量把握」式温情劝导。

## 节奏公式

| 公式 | 特征 | 适用 |
|------|------|------|
| 快切 | 信息切换频繁、单段短 | 教程、好物种草 |
| 叙事 | 单线推进、情绪递进 | 剧情、口播 |
| 循环 | 重复结构强化记忆 | 对比、测评 |
| 递进 | 层层深入放大痛点/利益 | 痛点解决型、科普 |

## 信息切换间隔建议

- 短视频（≤60s）：3–8s 一切。
- 中长（60–180s）：15–45s 一段，单段不超 45s 防流失。
- `rhythm.info_switch_interval` 按实际填写（如 `"5-8s"`、`"30-45s"`）。

## 情绪曲线模板

用 `→` 串联，示例：
- 痛点型：`担忧→惊恐质疑→找到解法→安心信赖`
- 种草型：`好奇→认同→渴望→下单冲动`
- 剧情型：`代入→冲突→反转→释然`
- 教程型：`好奇→跟随→成就感→分享欲`

## 爆款潜质自检清单（兜底生成强制套用）

兜底生成时，所选「目的×行业×受众×选题×产品」组合必须命中下列 **≥3 项**：

- [ ] **受众痛点真实**：选题对应受众高频、高情绪的真实痛点。
- [ ] **钩子有张力**：Hook 3 秒内能命中上表某钩子手法，且非平淡陈述。
- [ ] **有信息增量**：中段提供受众「不知道但有价值」的信息（数据、方法、对比、反常识）。
- [ ] **情绪有起伏**：情绪曲线非「平淡→平淡」，至少一次明显转折。
- [ ] **产品/解法闭环**：有产品时痛点→解法闭环完整；无产品时种草/涨粉闭环完整。
- [ ] **节奏不拖**：单段 ≤45s，信息切换间隔在建议区间内。

命中 < 3 项 → 重新拟定组合，不得直接产出。
